# リスタート相談室 ランディングページ

## 設置方法

1. index.htmlと同じディレクトリに「images」フォルダを作成してください。
2. 以下の画像ファイルを「images」フォルダに配置してください：
   - money-consultation.png
   - problems-image.png
   - service.png
   - moc.png
   - customer-voice-image.png

## 注意事項

- このHTMLファイルはTailwind CSSのCDNを使用しています。インターネット接続が必要です。
- オフラインで使用する場合は、Tailwind CSSをダウンロードして、ローカルで参照するように変更してください。
- 画像ファイルのパスが正しいことを確認してください。

## カスタマイズ

- HTMLファイルを編集して、テキストや画像、リンクなどを変更できます。
- Tailwind CSSのクラスを編集して、スタイルをカスタマイズできます。
